# Diagnostic Fixtures

This folder contains examples whose primary purpose is to exercise diagnostics,
warnings, solver failure artifacts, and runtime data-quality failure artifacts.

```text
error_messages/
  Parser, semantic, unit, workflow, equation, simulation, class, domain,
  component, port, and connection diagnostic fixtures.

data_quality/
  CSV policy and runtime data-quality fixtures, including parse failures,
  interpolation, constraint violations, and unsupported conversions.
```
