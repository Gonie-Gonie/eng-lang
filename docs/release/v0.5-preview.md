# v0.5-preview Release Notes

Status: ready as TimeSeries/statistics metadata preview. The later v1.0
hardening path materializes official CSV TimeSeries values, mean/max/p95
statistics, and trapezoidal integration in runtime artifacts.

## Scope

v0.5-preview adds the first engineering data-analysis metadata path:

```text
promoted table -> TimeSeries[Time] -> summary metadata -> integrate metadata -> result payload
```

Included:

```text
- workspace version bumped to 0.5.0
- summarize statement AST seed
- TimeSeries[Time] of HeatRate inference for Q_coil-style table expressions
- axis metadata for promoted tables and TimeSeries values
- StatsInfo metadata for summarize Q by [mean, max, p95]
- lazy summary cache key in the original v0.5 preview
- IntegrationInfo metadata for integrate(Q, over=Time)
- HeatRate sum lint W-STATS-SUM-001
- TimeSeries bytecode object record
- load_timeseries bytecode instruction
- TimeSeries VM object store entry
- result.engres statistics and integrations payload
- report.html axis/statistics/integrations sections
- heat_rate_sum warning example
- tests for TimeSeries metadata, VM TimeSeries execution, and HeatRate sum lint
```

## Artifact Additions

```text

0003|load_timeseries|Q_coil
```

`result.engres` can now include:

```text
object_store.timeseries_count
typed_payload.statistics
typed_payload.integrations
```

`review.json` can now include:

```text
axis_info
stats_info
integrations
```

## Known Limits

```text
- row-level numeric TimeSeries values
- mean/max/min/p95 numeric kernels
- time-weighted mean
- non-uniform time handling
- summary materialization
- broader statistics values in report cards beyond the v1.0 official CSV path
```
