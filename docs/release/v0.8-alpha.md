# v0.8-alpha Release Notes

Status: ready as minimal system/equation alpha.

## Scope

v0.8-alpha adds residual-only physical system metadata:

```text
system block -> parameter/state/input -> equation block -> eq unit check -> residual metadata
```

Included:

```text
- workspace version bumped to 0.8.0
- system block parser
- parameter/state/input parser
- equation block parser
- infix eq relation parser
- der() dimension handling
- HeatCapacity quantity seed
- J/K and kJ/K unit seeds
- equation unit consistency diagnostic
- diagnostic when equation blocks use ==
- simple residual metadata
- system_summary in review.json
- system_summary in report_spec.json
- system payload in result.engres
- System Equations section in report.html
- official simple system example
- eq boolean and equation unit mismatch examples
- compiler/report/CLI smoke tests for v0.8 behavior
```

## Commands

```bat
.\dev.bat setup

.\dev.bat ci
.\dev.bat package
```

```bat
target\debug\eng.exe run examples\06_simple_system\main.eng --entry main
target\debug\eng.exe view build\result\result.engres
type build\result\review.json
type build\result\report_spec.json
type build\result\result.engres
```

Error checks:

```bat
target\debug\eng.exe check examples\05_error_messages\eq_boolean.eng --review
target\debug\eng.exe check examples\05_error_messages\equation_unit_mismatch.eng --review
```
