# v1.0.1 Release Notes

Status: ready as a v1.0 hardening patch release.

## Scope

v1.0.1 keeps the v1.0 stable artifact contracts and adds the user-test tooling
needed before v1.1 uncertainty work begins.

Included:

```text
- workspace version bumped to 1.0.1
- native portable tester IDE shipped as eng-ide.exe
- package launcher shipped as eng-ide.bat
- native IDE smoke command: eng-ide.exe --smoke
- native IDE check/save/run/report workflow
- native IDE diagnostics from eng_compiler::check_source
- completion insertion for keywords, quantity kinds, units, and starter snippets
- compiler-derived symbol metadata panel
- secondary VS Code extension preview packaged as a VSIX
- official integrated HVAC user-test example
- integrated HVAC tutorial and native IDE guide
- release-check includes IDE validation and portable IDE smoke
```

## Portable Package

The package command creates:

```text
dist\englang-preview-v1.0.1-windows-x64.zip
dist\englang-preview-v1.0.1-windows-x64.zip.sha256
dist\release-manifest.txt
```

The unpacked package contains:

```text
eng.exe
eng-ide.exe
eng-ide.bat
examples\
stdlib\
docs\
tools\vscode-englang\
tools\englang-vscode-preview-1.0.1.vsix
README.txt
```

## User Test Flow

From the unpacked portable package:

```bat
eng.exe doctor
eng-ide.exe --smoke
eng-ide.bat
eng.exe run examples\official\03_integrated_hvac\main.eng --entry main
```

In the native tester IDE, open:

```text
examples/official/03_integrated_hvac/main.eng
```

Then use:

```text
Check -> edit/completion -> Save -> Run -> Open Report
```

## Known Limits

The native tester IDE is a release-preview user test tool, not a full LSP or
production IDE.

Deferred:

```text
- full LSP server
- production VS Code extension
- multi-file project model
- rich code actions and quick fixes
- adaptive, nonlinear, and multi-equation ODE/system solver
- uncertainty propagation
```
