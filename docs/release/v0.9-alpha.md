# v0.9-alpha Release Notes

Status: ready as portable demo hardening alpha.

## Scope

v0.9-alpha turns the preview from a repository-only developer build into a
portable Windows demo package:

```text
build release eng.exe -> assemble preview folder -> zip -> SHA256 -> clean-folder smoke
```

Included:

```text
- workspace version bumped to 0.9.0
- portable package zip named dist\englang-preview-v<version>-windows-x64.zip
- SHA256 checksum next to the portable zip
- package README.txt with target-PC smoke commands
- package-smoke wrapper command
- package-smoke extraction under a path containing spaces and Korean characters
- packaged eng.exe doctor smoke from the extracted folder
- packaged CSV+plot example smoke
- packaged simple system example smoke
- eng test examples path smoke for Korean and space-containing source/build paths
- README, roadmap, acceptance checklist, and reproducibility docs updated for v0.9
```

## Commands

```bat
.\dev.bat setup
.\dev.bat doctor
.\dev.bat ci
.\dev.bat package
.\dev.bat package-smoke
```

The package command creates:

```text
dist\englang-preview\
dist\englang-preview-v0.9.0-windows-x64.zip
dist\englang-preview-v0.9.0-windows-x64.zip.sha256
```

The unpacked package contains:

```text
eng.exe
examples/
stdlib/
docs/
README.txt
```

## Verified Behavior

`.\dev.bat ci` verifies:

```text
- official units example passes
- official CSV/report example passes
- official CSV+plot example produces report and PlotSpec artifacts
- official simple system example produces system report artifacts
- expected error examples produce diagnostics
- Korean and space-containing source/build path run smoke produces artifacts
- clippy with warnings denied
```

`.\dev.bat package-smoke` verifies:

```text
- release eng.exe builds
- portable preview folder is assembled
- zip and SHA256 files are written
- zip extracts into a path containing spaces and Korean characters
- packaged eng.exe doctor passes from the extracted folder
- packaged CSV+plot example creates result/report/PlotSpec artifacts
- packaged simple system example creates result/report artifacts
```
