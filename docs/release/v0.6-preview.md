# v0.6-preview Release Notes

Status: ready as PlotSpec/SVG/manifest preview.

## Scope

v0.6-preview replaces the static preview plot path with a PlotSpec-driven path:

```text
TimeSeries metadata -> PlotSpec v1 -> SVG renderer -> plot manifest -> report/view
```

Included:

```text
- workspace version bumped to 0.6.0
- PlotSpec v1 data model
- line plot seed
- axis unit labels
- deterministic preview points
- PlotSpec JSON serialization
- SVG rendering from PlotSpec
- plot_manifest.json
- plot spec and SVG hashes
- result.engres plot_spec_hash provenance
- `eng run` prints PlotSpec and manifest paths
- `eng view` lists plot_manifest.json
- `eng test` checks PlotSpec artifact generation
- PlotSpec JSON/SVG unit test
```
