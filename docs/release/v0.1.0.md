# v0.1.0 Release Notes

`v0.1.0` is a clean initial portable release for the parts of EngLang that are
ready to test as user-facing workflows.

It is not a solver release. Solver/system examples are source-repository
implementation fixtures under `examples/advanced_solver` and
`examples/internal`, not portable package tutorials. Do not use this release to
claim general ODE, DAE, nonlinear, adaptive, state-space, or multi-domain
component solving.

## Public Scope

This release supports:

- `eng.exe doctor`, `check`, `run`, `build --standalone`, and `view`.
- Typed CSV promotion through official examples.
- Unit-aware HeatRate calculations.
- TimeSeries summary statistics and integration metadata on the documented
  CSV/report path.
- PlotSpec v1 line and histogram output, SVG generation, and plot manifests.
- Review/result/report artifacts: `result.engres`, `review.json`,
  `report_spec.json`, `report.html`, `plot_spec.json`, `plot_manifest.json`,
  and `timeseries.svg`.
- Explicit `print`, structured `log`, summary CSV export, file-output manifest,
  process-result records, and local test-result records where covered by
  official examples.
- Standalone packaging for `examples/official/01_csv_plot/main.eng`.
- Native IDE smoke path and packaged `eng-lsp.exe --smoke`.
- Curated PDF user guide and language grammar guide.
- Optional VS Code extension package.

## Official Release Examples

The public first-user examples for this release are under `examples/official`.
Advanced solver smoke fixtures live under `examples/advanced_solver`; they are
regression coverage, not a broad solver claim.

Recommended first test:

```bat
eng.exe run examples\official\01_csv_plot\main.eng --save-artifacts
```

Additional public examples cover functions/imports, print/export, command
`where`/`with`, path policy, read-only IO, write/output manifests, explicit file
operations, structured logs, process results, local test assertions, and class
object authoring metadata.

## Source-Repository Fixtures

The source repository also keeps advanced and internal examples for IDE/LSP
development smoke, diagnostics, regression checks, and future-track inspection.
They are intentionally outside the portable package first-user example set.

Repository-only fixtures include:

- `examples/advanced_solver/*`
- `examples/internal/02_simple_system`
- `examples/internal/03_integrated_hvac`
- `examples/internal/17_measured_vs_simulated`
- uncertainty and data-driven modeling seeds
- domain/component fixtures
- state-space, adaptive, residual, and component-solver seeds

These are not public tutorials or release-supported solver workflows.
