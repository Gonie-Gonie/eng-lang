# v0.2-preview Release Notes

Status: ready as semantic skeleton preview

## Scope

v0.2-preview builds on v0.1 by adding semantic data structures that prepare the compiler for IDE/LSP support and stricter type checking.

Included:

```text
- workspace version bumped to 0.2.0
- ExpectedType internal API skeleton
- quantity completion data table skeleton
- inferred declaration hover hint data
- review.json expected_types section
- review.json hover_hints section
- review.json type_info section
- review.json unit_derivations section
- report.html hover hint summary
- report.html type info and unit derivation summary
- .engbc semantic summary counts
- ambiguous power quantity warning refinement
- dimensionless + physical subtraction diagnostics
- expected-context dimensionless diagnostic coverage
- official ambiguous quantity warning example
```
