# v0.2-preview Release Notes

Status: ready as the second public preview package after local release checks.

This release keeps the language and artifact formats in preview, but it closes
the IDE/documentation hardening slice and adopts the integrated language
philosophy for future work.

## Scope

`v0.2-preview` is a user-testable preview package for typed engineering data
workflows, native IDE inspection, and reproducible report/review artifacts.

Included:

```text
- eng.exe doctor/check/run/build/view
- top-level execution as the default root workflow
- root args block, importable const, pure scalar fn, and relative file imports
- parenthesis-light command syntax for built-in workflow verbs
- where local calculation blocks and with option/display blocks
- unit-aware print interpolation
- explicit summary CSV export under build/result
- typed CSV promote through official examples
- unit-aware TimeSeries calculations and statistics
- PlotSpec v1, SVG output, report.html, review.json, and report_spec.json
- basic standalone package execution
- native tester IDE with denser workspace, multi-file editing, variables,
  artifacts, runtime summaries, and plot inspection
- experimental eng-lsp.exe smoke/snapshot path
- packaged VS Code extension preview
- curated user PDF guide and language grammar PDF
```

New planning/design scope:

```text
- integrated typed data analysis + system simulation philosophy
- side-effect and general programming support policy
- class/domain-object track recorded as planned future work
- general programming/side-effect track recorded as planned future work
```

This is preview software:

```text
The language and artifact formats are not stable.
Future-track implementation seeds may exist on main but are not public support
claims.
```

## Current Boundaries

Public preview workflows:

```text
- typed CSV promote
- unit-aware TimeSeries calculation
- statistics and integration metadata
- function/import policy for pure scalar helpers
- command-style built-in workflow verbs
- print/export summary artifacts
- PlotSpec/SVG output
- runtime review/report objects and saved artifacts
- basic packaged execution
- native tester IDE
```

Future tracks, not public-supported in this release:

```text
- first-class Summary objects
- project-wide display unit policy blocks
- broad file I/O and side-effect profiles
- class/domain objects as executable language features
- uncertainty semantics beyond smoke examples
- data-driven modeling/ML semantics beyond smoke examples
- full persistent LSP/editor platform
- native JIT/AOT code generation or speedup claims
- numeric multi-domain component simulation
- domain package registry
```
