# v0.7-alpha Release Notes

Status: ready as basic report/review artifact alpha.

## Scope

v0.7-alpha hardens the review/report artifact path:

```text
compiler review metadata -> review.json
runtime report model -> report_spec.json
HTML report + PlotSpec/SVG artifacts
```

Included:

```text
- workspace version bumped to 0.7.0
- review_schema_version in review.json
- variable_table in review.json and report_spec.json
- inferred_declaration_table in report_spec.json
- unit_conversion_table in review.json and report_spec.json
- schema_summary in review.json and report_spec.json
- warning_list in review.json and report_spec.json
- plot_manifest section in review.json
- plot manifest path/hash section in report_spec.json
- report_spec_hash provenance in result.engres
- `eng run` prints report_spec.json
- `eng view` lists report_spec.json
- `eng test` checks report_spec.json generation
- v0.7 compiler/report unit tests
```

## Commands

```bat

```

Useful manual checks:

```bat
target\debug\eng.exe run examples\04_plotting\main.eng --entry main
target\debug\eng.exe view build\result\result.engres
type build\result\review.json
type build\result\report_spec.json
type build\result\plots\plot_manifest.json
```

PowerShell JSON checks:

```powershell
Get-Content build\result\review.json -Raw | ConvertFrom-Json | Select-Object format, review_schema_version
Get-Content build\result\report_spec.json -Raw | ConvertFrom-Json | Select-Object format, report_schema_version
```
