# v1.0.2 Release Notes

Status: ready as a tester IDE quality patch.

## Scope

v1.0.2 fixes the first user-test issues found in the v1.0.1 portable tester
IDE release.

Included:

```text
- workspace version bumped to 1.0.2
- eng-ide.exe release build uses the Windows GUI subsystem to avoid a separate console window
- eframe default fonts are enabled so text, buttons, and labels render visibly
- native IDE applies an explicit light theme for readable controls
- eng-ide.bat is removed from the portable package
- package smoke verifies the user guide PDF is present
- release-check creates a concise PDF user guide without external conversion tools
- GitHub Release publishes the PDF guide as a release asset
```

## Portable Package

The package command creates:

```text
dist\englang-preview-v1.0.2-windows-x64.zip
dist\englang-preview-v1.0.2-windows-x64.zip.sha256
dist\englang-user-test-guide-v1.0.2.pdf
dist\release-manifest.txt
```

The unpacked package contains:

```text
eng.exe
eng-ide.exe
docs\EngLang_User_Test_Guide.pdf
examples\
stdlib\
docs\
tools\vscode-englang\
tools\englang-vscode-preview-1.0.2.vsix

```

## User Test Flow

From the unpacked portable package:

```bat
eng.exe doctor
eng-ide.exe --smoke
eng-ide.exe
eng.exe run examples\official\03_integrated_hvac\main.eng --entry main
```

Read the bundled PDF first:

```text
docs\EngLang_User_Test_Guide.pdf
```

## Verified Behavior

`.\dev.bat release-check` verifies:

```text
- Rust tests and EngLang examples
- native IDE smoke
- docs-check
- IDE extension manifest check
- artifact schema/golden checks
- package-smoke from a Korean and space-containing path
- user guide PDF generation and packaging
- zip and SHA256 verification
```

## Known Limits

The native tester IDE remains a release-preview user test tool, not a full LSP
or production IDE. It is intended to make v1.1 feature testing easier before
the deeper IDE/LSP milestones.
