# v0.3-preview Release Notes

Status: ready as CSV data-boundary preview

## Scope

v0.3-preview adds the first schema and CSV promotion path. It does not execute full typed tables yet, but it validates CSV headers and records data provenance.

Included:

```text
- workspace version bumped to 0.3.0
- parser context for schema constraints and missing policy blocks
- schema symbol table
- schema columns with unit and index metadata
- schema constraints seed metadata
- missing policy seed metadata
- promote csv expression analysis
- source-relative CSV path resolution
- CSV header validation
- CSV source hash provenance
- missing required column diagnostic
- missing policy unknown column diagnostic
- review.json schema and csv_promotions sections
- report.html schema and CSV promotion sections
- result.engres data hash provenance
- missing CSV column error example
```
