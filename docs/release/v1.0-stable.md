# v1.0-stable Release Notes

Status: ready as stable core release.

## Scope

v1.0-stable closes the first execution arc:

```text
typed CSV boundary -> unit-aware calculations -> TimeSeries statistics -> PlotSpec/report -> minimal system metadata -> packaged runner
```

Included:

```text
- workspace version bumped to 1.0.0
- official units, CSV/report, CSV+plot, and simple system examples pass
- stable `eng run` artifact path through bytecode v1 and result.engres v1
- row-level runtime table pages for the official CSV example
- computed mean/max/p95 statistics and trapezoidal HeatRate integration for
  the official coil TimeSeries path
- schema constraint and missing policy execution status for the official CSV path
- review.json, report_spec.json, report.html, PlotSpec, SVG, and plot manifest generation
- CSV-derived PlotSpec points with title and y-axis unit options applied
- minimal system/equation residual metadata
- portable zip package and SHA256 checksum
- portable package smoke under a path containing spaces and Korean characters
- runnable standalone bundle from `eng build --standalone`
- standalone run.bat execution that produces report and PlotSpec artifacts
- `.engpkg` package metadata promoted to `engpkg-stable-1`
- standalone lock file records runtime/compiler/bytecode/result/report/plot versions
```

## Stable Boundaries

Stable for v1.0:

```text
- v8/v9 fast `=` syntax policy
- typed CSV promote validation
- entry-based run/build policy
- bytecode v1 text artifact
- result.engres v1 typed payload
- review.json/report_spec.json/report.html review path
- PlotSpec v1 and native SVG plot output
- minimal system/equation residual metadata
- portable package and packaged runner execution
```

Important qualification:

```text
v1.0 is stable as an artifact-contract and packaged-runner release. It does not
claim a general-purpose expression VM for arbitrary table formulas, generated
Args-derived flag binding, or system simulation.
```

Deferred:

```text
- numeric ODE/system solver
- broad table expression execution beyond the supported official CSV coil path
- duration_above and broader statistics kernels
- missing value interpolation
- runtime binding for Args-derived flags
- component/connect graph
- uncertainty propagation
- ML package
- LSP/VS Code extension
- optimized AOT standalone executable
- package registry and domain package ecosystem
```

## Release Workflow

v1.0-stable was tagged before the GitHub release workflow existed. To publish
or refresh the GitHub Release assets after this workflow is merged, manually run
`.github/workflows/release.yml` with:

```text
tag_name = v1.0-stable
```

For the public SemVer GitHub Release, use tag `v1.0.0`. The release workflow
falls back to this file for `v1.0.0` release notes.

Post-release hardening is tracked in
[v1.0 gap audit](../development/05_v1_0_gap_audit.md).
