# v0.1-preview Release Notes

Status: ready as preview foundation

Commit baseline:

```text
6f82116 Bootstrap EngLang v0.1 preview
```

## Scope

v0.1-preview establishes the repo, development environment, compiler frontend skeleton, runtime artifact flow, and documentation baseline needed to continue toward v0.2.

Included:

```text
- Rust workspace with eng_cli, eng_compiler, eng_runtime, eng_report
- repo-local Windows setup through dev.bat
- pinned Rust toolchain
- Python-free core preview path
- eng.exe command skeleton
- source span model
- lexer token model
- parser AST skeleton
- semantic skeleton for v8 diagnostics
- .engbc preview artifact
- .engres preview artifact
- review.json
- report.html
- SVG preview plot
- official examples
- GitHub Actions workflow using dev.bat
- detailed development, architecture, CLI, language, roadmap, release docs
```

## Commands

```bat
.\dev.bat setup
.\dev.bat doctor
.\dev.bat ci
.\dev.bat package
```

Direct CLI:

```bat
target\debug\eng.exe doctor
target\debug\eng.exe check examples\05_error_messages\unit_mismatch.eng --review
target\debug\eng.exe run examples\04_plotting\main.eng
target\debug\eng.exe build examples\04_plotting\main.eng --standalone --profile repro
target\debug\eng.exe view build\result\result.engres
```
