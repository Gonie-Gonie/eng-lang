# Read CSV And Plot

Start from examples/official/01_csv_plot/main.eng.

1. Define a schema with typed columns and a DateTime index.
2. Promote the CSV with promote csv.
3. Derive a TimeSeries with unit-aware expressions.
4. Add a report block with a plot over Time.
5. Run with a command such as eng.exe run source.eng --save-artifacts.

Inspect review.json for schema promotion, TimeSeries metadata, and plot artifact
paths.
