# Versioned Language Policy Summary

This page keeps the historical `language_v8.md` link target stable, but it is
no longer the detailed policy source. Current language behavior is documented in
the focused reference pages below.

## Current Policy Sources

- [Fast assignment](fast_assignment.md): `name = expr` local binding and
  assignment rules, plus the unsupported `:=` diagnostic.
- [Dimensionless and unit policy](dimensionless.md): dimensionless arithmetic,
  explicit-unit requirements, and ambiguous quantity diagnostics.
- [Syntax and grammar](syntax.md): current executable syntax, examples, and
  diagnostic catalog links.

## Historical Note

Older planning documents used v8/v9 labels for language-policy checkpoints.
Those labels are historical planning markers, not public release names. Current
release claims should come from [project status](../../current/status.md),
[version plan](../../current/version_plan.md), and the focused language
reference pages above.
