# v0.1-preview Release Notes

Status: published as the first public preview prerelease.

This release restarts the public EngLang version line. Earlier high-numbered
release names are not part of the current public release sequence.
This was not a `v0.1.0` release: the workspace package version was
`0.1.0-preview`, and the public label was `v0.1-preview`.

## Scope

`v0.1-preview` is a user-testable preview package for the current
data-to-report workflow.

Included:

```text
- eng.exe doctor/check/run/build/view
- typed CSV promote through official examples
- unit-aware TimeSeries calculations
- TimeSeries statistics and integration metadata
- PlotSpec v1 and SVG output
- runtime result/report/review/PlotSpec objects and saved artifacts
- basic standalone package execution
- native tester IDE eng-ide.exe
- experimental eng-lsp.exe smoke/snapshot path
- packaged VS Code extension preview
- curated user PDF guide
```

This is preview software:

```text
The language and artifact formats are not stable.
Future-track implementation seeds may exist on main but are not public support
claims.
```

## Current Boundaries

Public preview workflows:

```text
- typed CSV promote
- unit-aware TimeSeries calculation
- statistics and integration metadata
- PlotSpec/SVG output
- runtime review/report objects and saved artifacts
- basic packaged execution
- native tester IDE
```

Future tracks, not public-supported in this release:

```text
- uncertainty semantics beyond smoke examples
- data-driven modeling/ML semantics beyond smoke examples
- full persistent LSP/editor platform
- native JIT/AOT code generation or speedup claims
- numeric multi-domain component simulation
- domain package registry
```

## Verification

Run:

```bat
.\dev.bat release-check
```

The release check validates CI, docs snippets, artifact golden baselines,
package assembly, clean-folder package smoke, IDE smoke, LSP smoke, standalone
package execution, checksum verification, and release manifest generation.
