# v0.5-preview Release Notes

`v0.5-preview` promotes the write/export hardening slice of the general
programming and side-effect policy. It makes generated outputs explicit,
repeatable, and reviewable through an output manifest.

EngLang remains preview software. Language syntax, runtime behavior, and
artifact schemas are not stable yet.

## Package Label

```text
v0.5-preview
```

Cargo/package version:

```text
0.5.0-preview
```

## Highlights

```text
- write text/json top-level output statements
- idempotent overwrite hardening for write/export outputs
- output_manifest.json for generated file artifacts
- review.json writes metadata
- official/12_write_output_manifest mini example
```

## Write And Export Scope

The supported preview forms are:

```eng partial
export summary to csv "outputs/summary.csv" {
    mean_Q as kW with ".2"
}
with {
    overwrite = true
}

write text "outputs/run_note.txt", notes_text
with {
    overwrite = true
}

write json "outputs/energy.json", E_coil
with {
    overwrite = true
}
```

Output targets are constrained under `build/result`. Absolute paths and parent
directory traversal are rejected for generated outputs.

Existing output behavior:

```text
- missing file: write normally
- existing identical file: accepted as an idempotent rerun
- existing different file: requires with { overwrite = true }
```

## Output Manifest

Runs now produce:

```text
build/result/output_manifest.json
```

The manifest records:

```text
format
runtime_version
source_path
artifact_count
artifacts[].kind
artifacts[].path
artifacts[].hash
```

When `--save-artifacts` is used, result/report/PlotSpec/SVG artifacts are
listed alongside CSV export and write outputs.

## Official Example

Run:

```bat
eng.exe run examples\official\12_write_output_manifest\main.eng --save-artifacts
```

Expected generated outputs include:

```text
build/result/outputs/summary.csv
build/result/outputs/run_note.txt
build/result/outputs/energy.json
build/result/output_manifest.json
```

## Deferred

```text
- copy/move/delete
- mkdir/list
- broad filesystem mutation outside generated outputs
- external process and network effects
- structured JSON/TOML values
```

The immediate next target is `v0.6-preview`: explicit copy/move/delete
side-effect policy.
